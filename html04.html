이미지 관련 요소

이미지 5개 이상 - jpg, png, gif, wedp, svg (웹 접근성)

.svg 확장자로 설정하면 확대해도 픽셀이 깨지지 않는다.


동영상 3개 이상 - mp4, webm, ogv

  • video 요소의 태그 속성

    src : 해당 비디오 파일의 경로(위치와 파일명)

    poster : 대기화면(썸네일) 이미지의 파일 경로

    controls : 비디오 재생 제어 패널 표시

    autoplay : 자동재생

    muted : 무음처리(자동 재생시 반드시 필요함)

    loop : 반복 재생

    width : 폭, height : 높이

    • 브라우저 마다 재생가능한 파일(크로스 브라우징)

    mp4, webm, ogv = 파일이 브라우저 마다 다른 재생 기능 제공 (표준 파일 타입) - 각기 다른 확장자 파일 준비

    브라우저MP4WebMOgv
    Internet ExplorerYESNoNo
    ChromeYESYESYES
    FirefoxYESYESYES
    SafariYESNoNo
    OperaYESYESYES

  • iframe은 width, height, src가 기본속성(최소한 작성해야하는 태그)이다.
